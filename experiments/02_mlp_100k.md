# Experiment 2: MLP at 100K Examples

## Goal

Scale up the MLP memorization task to 100K examples. Test whether the model
can still memorize with a much larger dataset.

## Setup

- **n_examples**: 100,000
- **input_len**: 1,000, **output_len**: 1, **vocab**: 2
- **Model**: 1-layer MLP, d_ff=1024
- **Sampler**: power-law with beta in {0.5, 1.5}
- **Steps**: 200K, **batch_size**: 512

## Configs

- `powerlaw_N100k_mlp_b0.5.yaml`
- `powerlaw_N100k_mlp_b1.5.yaml`

## Results

With 100K examples, the model achieved partial memorization. High-frequency
examples (low rank under power-law) were learned first, with a clear
frequency-dependent learning order.
